# Support

Need help with rivr?

- **Start with `rivr doctor --json`** — it checks credentials, connectivity, secret-file
  perms, and affiliate state, and prints a fix for each failure.
- **Read the docs:** https://rivr.sh (or `rivr agent` for the embedded agent guide, and
  `rivr schema` for the machine-readable command/exit-code spec).
- **Questions / ideas:** open a [GitHub Discussion](https://github.com/rnwolfe/rivr/discussions).
- **Bugs:** open an [issue](https://github.com/rnwolfe/rivr/issues/new/choose) using the form
  (include `rivr version`, OS, the exact command, and `--json` output).
- **Security:** do **not** file a public issue — see [SECURITY.md](./SECURITY.md).

This is a community project maintained on a best-effort basis; there is no commercial SLA.
